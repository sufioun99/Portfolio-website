# MD Abu Sufioun Nasim - Portfolio

A modern, responsive portfolio website showcasing the skills, projects, and achievements of MD Abu Sufioun Nasim.

## Features

- **Dual Template Design**: Switch between two professional templates
- **Dark/Light Theme Toggle**: User-friendly theme customization
- **Responsive Layout**: Optimized for all screen sizes
- **Smooth Animations**: Professional hover effects and transitions
- **Section Navigation**: Easy access to all portfolio sections

## Technologies Used

- **HTML5**: Semantic structure
- **CSS3**: Modern styling with CSS variables and animations
- **JavaScript**: Interactive functionality
- **Font Awesome**: Icon library
- **Google Fonts**: Inter font family

## Sections

1. **Home**: Hero section with introduction
2. **About**: Personal information and expertise
3. **Skills**: Technical skills categorized by expertise
4. **Work**: Project showcase with detailed descriptions
5. **Education & Certifications**: Academic background and certifications
6. **Achievements**: Awards and recognitions
7. **Contact**: Contact information

## Running the Portfolio

### Local Development

1. Clone or download the repository
2. Open a terminal in the project directory
3. Run a local server using Python:
   ```bash
   python -m http.server 8000
   ```
4. Open your browser and navigate to `http://localhost:8000`

### Direct Opening

Alternatively, you can directly open `index.html` in your web browser.

## Deployment

### GitHub Pages

1. Create a new repository on GitHub
2. Upload all portfolio files
3. Go to Repository Settings > Pages
4. Select the main branch and root directory
5. Your portfolio will be available at `https://yourusername.github.io/repositoryname/`

### Netlify/Vercel

1. Sign up for a free account on Netlify or Vercel
2. Connect your GitHub repository
3. Follow the deployment instructions
4. Your portfolio will be live within minutes

### Custom Domain

You can connect your custom domain by following the domain settings in your chosen hosting platform.

## Customization

### Updating Content

- **Personal Information**: Edit the content in `index.html`
- **Projects**: Modify the project cards in the work section
- **Skills**: Update the skills grid in the skills section
- **Certifications**: Add/remove certificates in the achievements section

### Styling

- **Colors**: Modify CSS variables in `styles.css`
- **Fonts**: Change Google Fonts link in `index.html`
- **Layout**: Adjust grid and flex properties in `styles.css`

### Adding New Projects

1. Create a new HTML file for your project (e.g., `my-project.html`)
2. Add the project content with proper styling
3. Create a project card in the work section of `index.html`
4. Link the project card to your new HTML file
